Title: Detection of Attempting Login Using Machine Learning

This project tests the hypothesis that account takeover attempts can be detected from behavioral, network, and context-based login features extracted from large-scale authentication telemetry.

Dataset: Kaggle RBA Dataset (synthetic records from a Norwegian SSO service, 2020-2021). Models trained: Logistic Regression, Random Forest, and XGBoost.
